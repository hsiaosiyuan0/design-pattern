# 地理附录：古今地名与行旅路线

这份附录服务于两件事：

- 帮读者建立《乱世里的设计模式》的空间感
- 帮章节配图提供“位置图/地图图”的依据

这套故事不追求严格史学考据，但地理关系必须大体可靠，否则连载读下来会失去方向感。

## 核心原则

- 每一回至少要让读者知道“事情大概发生在哪”
- 每跨地区移动一次，最好给出前后位置关系
- 地图图不必复杂，但必须清楚标出古地名和今地名

## 地图骨架规范

后续所有地图都不再按“补几个地标”的方式处理，而是按完整骨架来检查。

地图骨架分为五层：

### 一、水系骨架

水系决定方向感、交通感和区域分割感。

系列总地图优先保留：

- 黄河
- 长江
- 淮河
- 大运河

章节地图按范围择优保留：

- 章节附近的主干河流
- 与叙事有关的渡口、水路、运河节点

说明：

- 黄河是河东、中原、汴梁叙事的北方骨架
- 长江是江淮、吴越、江南叙事的南方骨架
- 淮河决定淮南与江淮的区域边界感
- 大运河适合表现汴梁、江淮、东南之间的联通性

### 二、山脉与地形骨架

山地决定阻隔、险要、边关和路线难度。

系列总地图优先保留：

- 太行山
- 秦岭
- 大别山
- 四川盆地边缘山地

章节地图按场景择优保留：

- 雁门关附近山口
- 河东北境关隘地形
- 入蜀山道

说明：

- 太行山是河东与中原之间最重要的边界感来源
- 秦岭帮助读者理解中原与西南、西北空间的分隔
- 大别山有助于表现淮南、江淮与中原过渡带
- 四川盆地边缘山地决定蜀地“易守难出”的空间印象

### 三、区域板块骨架

地图不只是标城市，还要让读者知道每座城属于哪一片天下。

系列总地图建议保留：

- 河东
- 中原
- 江淮
- 淮南
- 吴越
- 蜀地
- 江南

说明：

- 区域板块比行政边界更重要
- 读者需要先知道“大致哪一块”，再知道“具体哪一城”

### 四、城市与政治节点骨架

这是叙事真正落脚的点。

系列总地图建议保留：

- 太原 / 晋阳
- 潞州
- 雁门
- 汴梁 / 开封
- 淮南核心节点
- 吴越核心节点
- 蜀地核心节点
- 江南晚年节点

说明：

- 节点不必过密
- 节点必须服务故事推进，而不是做成百科地图

### 五、行动与叙事骨架

这是《乱世里的设计模式》区别于普通古地图的地方。

建议保留：

- 主角沈策的主要迁移路线
- 卷次分段
- 关键事件发生地
- 前后章节之间的空间关系

说明：

- 没有故事线的地图只是背景
- 有了行动骨架，地图才会真正变成阅读导览

### 河东

- 古代语境：河东节度使辖境
- 大致今地：山西中北部，以太原一带为核心
- 叙事功能：故事起点，边地军府，适合战争、军议、烽火与筑城

### 太原

- 古名常见：晋阳、并州、太原
- 今地：山西太原
- 叙事功能：沈策初入军府的核心城市

### 潞州

- 今地大致对应：山西长治一带
- 叙事功能：前线告急、援军与粮道压力的重要节点

### 雁门

- 今地大致对应：山西代县、雁门关一带
- 叙事功能：观察者模式、边关与烽火体系的天然场景

### 汴梁

- 古名常见：开封、汴州、汴梁
- 今地：河南开封
- 叙事功能：中枢、官署、诏令、责任链、外观、中介者等制度型章节中心

### 淮南

- 古代语境：淮南节度区或泛指淮河以南、江淮之间
- 今地大致范围：安徽中东部、江苏西部部分区域
- 叙事功能：藩镇势力深，适合代理、借粮、权力边界与门禁秩序

### 江淮

- 今地大致范围：江苏中部、安徽东部、长江与淮河之间
- 叙事功能：水陆转运、桥接、抽象工厂、后勤制度

### 吴越

- 今地大致范围：浙江、苏南一带，以杭州周边为代表
- 叙事功能：番邦使节、海贸文书、适配器模式的异制度来源

### 蜀地

- 今地：四川盆地，以成都平原为核心
- 叙事功能：平定后旧档如山，适合原型、迭代器、备忘录

### 江南

- 今地泛指：长江下游南岸诸地，偏苏南、浙北、江西东北等文化空间
- 叙事功能：晚年收束、整理旧制、访问者与解释器

## 建议地图样式

### 章节位置图

- 用途：放在每一回开头或楔子后
- 内容：只标当前章节主要地点
- 标注方式：古名在前，今名在后

示例：

- `汴梁（今河南开封）`
- `太原 / 晋阳（今山西太原）`
- `蜀地（今四川成都平原一带）`

### 路线图

- 用途：当沈策跨地区移动时使用
- 内容：上一回地点、本回地点、行旅方向
- 表达方式：地图上用细线或箭头连接

示例路线：

- 河东 -> 汴梁
- 汴梁 -> 淮南
- 淮南 -> 雁门
- 汴梁 -> 蜀地
- 蜀地 -> 江南

### 天下总览图

- 用途：系列总图
- 内容：河东、汴梁、淮南、江淮、吴越、蜀地、江南六大锚点
- 必带骨架：主要水系、主要山脉、区域板块、政治节点、故事线
- 目标：帮助读者理解整套故事并不是发生在同一座城里

### 系列总地图最低要求

至少应同时具备：

- 黄河、长江、淮河、大运河中的主要水系
- 太行山，以及必要时的秦岭或蜀地山势
- 河东、中原、江淮、吴越、蜀地、江南等区域板块
- 太原 / 晋阳、汴梁 / 开封、雁门、潞州等关键节点
- 一条能看出卷次推进的故事主线

### 章节位置图最低要求

至少应同时具备：

- 当前章节的核心地点
- 所属区域板块
- 最近的骨架水系或山脉
- 与上一章节地点的大致空间关系

### 路线图最低要求

至少应同时具备：

- 起点
- 终点
- 经过区域
- 明确方向感
- 路线与地形之间的关系

## 地图标注建议

- 标注优先写法：`古名（今地）`
- 若古名有多个常见叫法，可写：`汴梁 / 开封（今河南开封）`
- 不必追求县级精确，经纬度感比行政区精确更重要
